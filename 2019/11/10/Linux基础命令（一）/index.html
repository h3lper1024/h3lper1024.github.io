    Linux基础命令（一）

    1.命令前缀

    2.pwd命令

    ​ 查看当前光标所在的目录

    3.cd命令

    ​ cd - ：打开上次访问的目录

    ​ cd ~：打开当前用户家目录

    ​ cd ..：打开上级目录

    4.ll命令

    ​ ls：显示目录中的文档

    ​ -l： 显示目录文档详情

    ​ -a：显示文档中的隐藏文档

    ​ -h：以易阅读方式显示

    ​ -rt：以时间倒叙排序

    ​ ls -l 等价于ll

    5.查看命令帮助

    ​ --help

    6.创建文件夹

    ​ mkdir A：创建A文件夹

    ​ mkdir -p A/B/C：级联创建A/B/C

    ​ mkdir A B C：同级创建A、B、C

    7.mv命令

    ​ mv a.log A：将a.log移动到A文件夹下

    ​ mv a.log A/b.log：将a.log移动 到A文件夹下，并改名为b.log

    8.cp命令

    ​ cp a.log A：将a.log复制到A文件夹下

    ​ cp a.log A/b.log：将a.log复制到A文件夹下，并改名为b.log

    9.如何创建一个空文件 或者把一个文件设置为空

    ​ touch a.log

    ​ echo "" > a.log 慎用，文件大小不是0

    ​ cat /dev/null > a.log

    10.如何查看文件内容

    ​ cat ：显示全部文件内容

    ​ more ：支持空格键往下翻页，不能回退，按Q退出

    ​ less：支持上下翻页，按上下键翻页，按Q退出

    ​ 文件内容较少时，可以使用cat more less命令来查看，当查看如日志等大文件时，可采用实时查看的方式

    11.实时查看文件内容

    ​ tail -f a.log

    ​ tail -F a.log等价于 -f +retry

    ​ flume exec source 切记使用 -F

    12.如何定位ERORR文件

    1.当文件较小时

    ​ 可使用工具上传到Windows，然后搜索查看

    ​ 工具准备：yum install -y lrzsz

    ​ 上传到Windows：sz a.log

    2.当文件较大时

    ​ cat a.log | grep ERROR 只输出ERROR所在行

    ​ cat a.log | grep -A 10 ERROR 输出ERROR及前10行消息

    ​ cat a.log | grep -B 10 ERROR 输出ERROR及前10行消息

    ​ cat a.log | grep -C 10 ERROR 输出ERROR及前后各10行消息
