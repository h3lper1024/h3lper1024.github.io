    Linux基础命令（三）

    1.用户/用户组

    • 查看与用户相关的命令：

      ll /usr/sbin/user*

      共有：useradd,userdel,usermod,usernetctl

    • 查看与用户组相关的命令：

      ll /usr/sbin/group*

      ​ 共有：groupadd,groupdel,groupmems,groupmod

    2.创建用户

    ​ useradd xxx

    ​ 创建用户会对应的创建用户组，并且设置用户主组，创建home/xxx文件夹

    3.删除用户

    ​ userdel xxx

    ​ 当删除用户当前组只有这一个用户，会删除用户组，但不会删除home/xxx

    4.查看用户

    ​ id xxx

    ​ 显示用户，用户组，用户主组

    uid=1002(xxx) gid=1002(xxx) groups=1002(xxx)

    5.模拟用户样式丢失

    ​ 正常：[user@hostname ~]

    ​ 异常：-bash-4.2$

    ​ 原因：/home/user 文件夹下缺少.bash_logout,.bash_profile,.bashrc

    ​ 修正：

    cp /etc/skel/.* /home/user/

    chown user:user .bash*

    6.创建一个新的用户组，并将user添加到该组

    groupadd group1

    cat /etc/group |grep bigdata

    usermod -a -G bigdata ruoze

    7.将user主组设置为group1

    usermod --gid group1 user

    8.设置密码

    passwd user

    9.切换用户

    su user

    su - user 代表切换该用户的家目录，且执行环境变量文件

    .bash_profile .bashrc
    su user 不执行 执行
    su - user 执行 执行

    10.普通用户获取root的最大权限

    vi /etc/sudoers

    user ALL=(root) NOPASSWD:ALL

    ​ 使用时要在命令前加sudo

    11.发生su - user切换用户失败

    ​ 查看/etc/passwd文件对应用户设置是否正确

    ​ 正确应该为：

    ​ user:x:1002:1003::/home/user:/bin/bash

    ​ 错误的示例：

    ​ ruoze:x:1002:1003::/home/ruoze:/bin/false

    ​ ruoze:x:1002:1003::/home/ruoze:/sbin/nologin

    12.文件/文件夹属性

    ​ drwxr-xr-x 2 root root 6 Nov 16 22:15 a.log

    第一个字母d：表示文件夹

    • d：文件夹
    • -：文件
    • l：链接

    rwx：权限

    • r：读权限 4
    • w：写权限 2
    • x：执行权限 1

    第一组：当前用户的权限

    第二组：当前用户所在用户组权限

    第三组：其他用户组权限

    ​ 其他详情：链接

    13.权限修改

    ​ chmod

    14.查看文件/文件夹大小

    • 文件:

      ll

      du -sh xxx.log

    • 文件夹

      du -sh xxx

    15.查找文件

    ​ find

    history 历史记录

    ps -ef 查看进程

    top 系统情况

    ​ 查找hadoop：

    ​ find / -name '*hadoop*'
