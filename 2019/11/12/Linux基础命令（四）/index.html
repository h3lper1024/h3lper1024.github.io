    Linux基础命令（四）

    1.vi

    ​ 三种模式：命令行模式、编辑模式、尾行模式

    ​ 对生产配置文件进行vi编辑属于高危命令，应该先进行备份

    1.1命令行模式常见命令

    dd 删除当前行

    dG 删除光标当前及以下的所有行
    ndd 删除光标当前及以下的n行

    gg 跳转到第一行的第一个字母
    G 跳转到最后一行的第一个字母
    shift+$ 行尾

    1.2粘贴

    ​ 必须在编辑模式下进行

    1.3查找ERROR

    ​ 在尾行模式输入/，输入关键字，按N翻页

    1.4设置行号

    ​ 在尾行模式

    set nu

    set nonu

    1.5清空内容

    cat /dev/null > xxx.log 完美

    echo ‘’ > xxx.log 会存在一个字节

    gg dG

    2.系统命令

    2.1查看磁盘状态

    df -h

    2.2查看内存状态

    free -m

    2.3查看机器负载

    top

    关注：load average: 0.00, 0.01, 0.05

    第一个数字：第1min负载

    第一个数字：第5min负载

    第一个数字：第15min负载

    不超过3/10

    3.查看、杀死进程

    3.1查看进程

    ps -ef | grep xxx

    结果：

    第一列：进程所属用户

    第二列：进程ID，PID

    第三列：进程父ID

    3.2杀死进程

    高危命令

    kill -9 PID
    kill -9 PID PID …
    kill -9 $(pgrep -f 匹配关键词) 杀死于关键字有关的所有进程

    4.查看进程的端口号

    netstat -nlp | grep PID （准确）

    或

    netstat -nlp | grep 进程名

    0.0.0.0:22 当前的ip

    :::22 当前的ip

    127.0.0.1:22 本地可以自己访问自己 window和其他服务器无法进行访问
    localhost:22 本地可以自己访问自己 window和其他服务器无法进行访问

    5.yum源安装

    5.1搜索

    yum search xxx

    5.2安装

    yum install -y xxx

    5.3卸载

    yum remove xxx

    6.rpm查看，卸载

    6.1rpm查看已安装软件

    rpm -qa | grep xxx

    6.2rpm卸载

    rpm -e xxxx 会检查依赖情况

    rpm -e - nodeps xxx 不检查依赖，直接卸载

    7.wget下载安装包

    wget xxx

    8.压缩/解压

    8.1tar文件
    8.1.1解压

    tar -zxvf xxx.tar.gz

    8.1.2压缩

    tar -zcvf xxx

    8.2zip文件
    8.2.1解压

    unzip xxx.zip

    8.2.2压缩

    zip xxx

    zip -r xxx.zip ./* 文件夹压缩，在文件夹里面

    zip -r xxx.zip xxx/* 文件夹压缩，在文件夹外面
