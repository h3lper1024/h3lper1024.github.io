      CREATE (DATABASE|SCHEMA) [IF NOT EXISTS] database_name
      [COMMENT database_comment]
      [LOCATION hdfs_path]
      [WITH DBPROPERTIES (property_name=property_value, …)];

      例：

      ​ 创建库，且指定位置

      CREATE DATABASE IF NOT EXISTS stu LOCATION /xxx;

    默认default库在HDFS的位置：

    ​ default ==> hdfs://usr/hive/warehouse

    ​ 即：${hive.metastore.warehouse.dir}

    指定LOCATION创建的库在HDFS的位置：

    ​ xxx ==> hdfs://user/hive/warehouse/xxx.db

    ​ 即：${hive.metastore.warehouse.dir}/dbname.db
