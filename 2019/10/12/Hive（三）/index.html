    Hive基础

    1.入门命令

    1. 启动Hive

      hive

    2. 查看所有数据库

      show databases；

    3. 切换数据库

      use xxx;

    4. 查看所有表

      show tables;

    5. 创建表

      create table stu(id int, name string, age int);

    6. 查看表结构

      desc stu;
      desc extended stu;
      desc formatted stu;（常用）

    7. 插入数据

      insert into stu values(1,’pk’,30);

    8. 查询

      select * from stu;

    9. 清屏

      !clear；

    10. 退出

      exit;

    2.Hive在HDFS上的位置

    ​ 以stu表为例

    默认：

    hdfs://ruozedata001:8020/user/hive/warehouse/stu

    配置：

    hive.metastore.warehouse.dir：/user/hive/warehouse
    stu：表的名字

    表的完整路径是: ${hive.metastore.warehouse.dir}/tablename

    3.查看Hive日志

    默认位置：/tmp/hadoop/hive.log

    查看日志位置配置

    cp hive-log4j.properties.template hive-log4j.properties
    hive.log.dir=${java.io.tmpdir}/${user.name}
    hive.log.file=hive.log
    ${java.io.tmpdir}/${user.name}/${hive.log.file}
    /tmp/hadoop/hive.log

    4.Hive配置

    查看所有配置参数：

    https://cwiki.apache.org/confluence/display/Hive/Configuration+Properties

    ​ 全局：$HIVE_HOME/hive-site.xml中
    ​ 临时/当前session：仅对当前session生效
    ​ 在Hive执行时：

    查看当前的属性：set key;
    修改当前的属性：set key=value;

    ​ 在Hive执行前：

    hive -hiveconf hive.cli.print.current.db=true

    5.Hive的交互式命令

    hive -e:

    不需要进入hive命令后，就可以跟上sql语句查询

    hive -e “select * from stu”

    hive -f:

    执行指定文件（内容是SQL语句）

    vi stu.sql

    hive -f stu.sql

    扩展：

    基于Hive的离线统计/数据仓库
    把SQL封装到shell脚本中去，使用hive -e “query sql…”
    定时调度：crontab
